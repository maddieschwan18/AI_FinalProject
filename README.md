# User Interface and AI Agent for Hyperdimensional m,n,k Games
This web application provides a clear user interface as well as an AI agent to play against for customizable two-dimensional, three-dimensional, and four-dimensional m,n,k games.

## How to Play
This application has been published via Replit, and may be accessed at https://grid-game-generator--g-mott.replit.app
- To set up a game, select an overall game mode (2D, 3D, or 4D) at the top of the settings panel
- Use the top sliders to select a number of rows, columns, layers, and hyperslices, as well as the number in a row needed to win
- By default, you can play against yourself, or locally with a friend, alternating placing an "X" or an "O" on the game space, like tic-tac-toe
- You may also choose to play against an AI agent. This agent evaluates every possible move and assigns each one a score to determine the optimal strategy. The "difficulty" level determines how likely the AI agent is to select the best possible move.
- Once a game is completed, the outcome is recorded on the scoreboard on the bottom. You can play as many games as you would like.
- You may reset the game or the scoreboard at any time.
